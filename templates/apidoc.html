Akses Database Terbuka

Integrasikan database jutaan jurnal ilmiah ke aplikasi Anda secara gratis, tanpa autentikasi, dan tanpa batas.

No API Keys JSON Output Rapid Response

Endpoint Utama (Base URL)

https://www.jurnalin.app/api/v1
GET

/search

Cari jurnal ilmiah di sistem kami

Query Parameters

q
Required. Kata kunci pencarian (topik, DOI, judul).
source
Filter sumber khusus: crossref, arxiv, doaj, semanticscholar.
page
Nomor halaman (default: 1).
year_from
Filter tahun awal riset.
year_to
Filter tahun akhir riset.
language
Filter bahasa: en, id, de, fr.
sort
Urutkan: relevance, year_desc, year_asc, citations.

Live Test

/search?q=deep+learning /search?year_from=2020&sort=citations /search?source=crossref

GET

/journal/:id

Ambil detail lengkap per dokumen

Gunakan parameter ID gabungan source_id. Contoh: arxiv_2301.12345 atau crossref_10.123/abc. Seluruh metadata penulis, sitasi, dan abstrak lengkap akan dikirimkan.

/api/v1/journal/arxiv_2301.12345 Test →

Usage Policy

API kami sepenuhnya gratis. Kami tidak memberlakukan rate-limit ketat secara default, namun mohon untuk menggunakan sistem secara bijak (fair use) agar ketersediaan database eksternal tetap terjaga.
